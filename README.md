# Personalized Diet Planner

This application helps create personalized diet plans based on BMI calculations and nutrient deficiency assessments. It provides tailored recommendations to help users achieve their health goals while addressing specific nutrient needs.

## Features

- BMI Calculator
- Nutrient Deficiency Assessment
- Personalized Diet Recommendations
- Additional Health Tips

## Installation

1. Clone this repository
2. Install the required dependencies:
```bash
pip install -r requirements.txt
```

## Running the Application

To run the application, execute:
```bash
streamlit run app.py
```

The application will open in your default web browser.

## How to Use

1. Enter your weight (in kg) and height (in meters)
2. Click "Calculate BMI" to see your BMI and category
3. Select any nutrient deficiencies you have been diagnosed with
4. Click "Generate Diet Plan" to receive personalized recommendations

## Note

This application provides general dietary recommendations. For specific medical advice, please consult with a healthcare professional or registered dietitian.
